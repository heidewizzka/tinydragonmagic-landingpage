# Press Kit Assets

Use this folder for press-ready material for Kingdom of Krawall.

- `downloads/`: ZIP bundles for journalists and creators.
- `key-art/`: capsule art, hero art, banners, and promotional images.
- `logos/`: game logos and studio logos, preferably transparent PNG/SVG.
- `screenshots/`: curated gameplay screenshots.
- `trailers/`: downloadable trailer files or trailer stills.
